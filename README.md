# Monkey Wrench
A Golang library for interfacing with Google Cloud Spanner.

Spanner, monkey wrench, get it? They're both tools, get it?...sigh

## Installation
Install the package as normal:
```bash
$ go get -u github.com/LUSHDigital/monkeywrench
```
